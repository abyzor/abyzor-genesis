**Free MIDI Pattern Generator for Electronic Music Producers**

Generate random melodies and MIDI patterns instantly for Techno, Trance, Psytrance, Drum & Bass, House and other electronic genres.

## Features

- Random MIDI pattern generation
- Scale and key selection
- MIDI Drag & Drop
- Fast workflow for idea generation
- Lightweight and easy to use
- Free to download
